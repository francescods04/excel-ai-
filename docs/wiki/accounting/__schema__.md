# Accounting Wiki Schema

## Purpose
This wiki contains accounting principles and financial statement knowledge. It covers GAAP, IFRS, and practical accounting for financial modeling.

## Conventions
- Distinguish between GAAP and IFRS when rules differ
- Cross-reference other pages with [[Page Name]]
- Include journal entries where helpful
- Cite authoritative sources (FASB, IASB) when possible

## Page Types
- **Principle**: accounting concepts and standards
- **Statement**: income statement, balance sheet, cash flow
- **Entry**: journal entries and adjustments
- **Ratio**: financial ratios and their calculation
- **Comparison**: GAAP vs IFRS differences

## Maintenance
When a new source is ingested:
1. Update relevant principle/statement pages
2. Add new pages if new standards or treatments appear
3. Update [[index]]
4. Log the update in [[log]]
