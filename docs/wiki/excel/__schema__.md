# Excel Wiki Schema

## Purpose
This wiki contains Excel mastery knowledge for financial modeling. It covers formulas, techniques, formatting, and best practices used in investment banking and corporate finance.

## Conventions
- Every formula example includes its Excel syntax
- Cross-reference other pages with [[Page Name]]
- Include keyboard shortcuts where relevant
- Note Excel version compatibility if relevant
- Update when new Excel features are released

## Page Types
- **Formula**: specific Excel formulas and use cases
- **Technique**: modeling techniques and workflows
- **Formatting**: visual standards and conditional formatting
- **Function**: deep dive into specific Excel functions
- **Best Practice**: conventions and standards

## Maintenance
When a new source is ingested:
1. Update relevant formula/technique pages
2. Add new pages if new concepts appear
3. Update [[index]]
4. Log the update in [[log]]
