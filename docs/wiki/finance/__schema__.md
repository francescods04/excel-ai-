# Finance Wiki Schema

## Purpose
This wiki contains institutional-grade financial modeling knowledge. It is the authoritative reference for the AI agents building DCF, WACC, LBO, and valuation models in Excel.

## Conventions
- All monetary values in millions ($M) unless specified otherwise
- All percentages as decimals in calculations (0.25 = 25%), formatted as 0.00%
- Cross-reference other pages with [[Page Name]]
- Every claim cites its source or reasoning
- Update pages when new sources are ingested

## Page Types
- **Concept**: explain a financial concept (e.g., [[Terminal Value]])
- **Model**: describe a complete model structure (e.g., [[DCF Model]])
- **Formula**: document Excel formulas and their financial meaning
- **Reference**: tables, ratios, benchmarks
- **Best Practice**: conventions and standards (e.g., [[IB Formatting Standards]])

## Maintenance
When a new source is ingested:
1. Update relevant concept pages
2. Add new pages if new concepts appear
3. Update [[index]]
4. Log the update in [[log]]
