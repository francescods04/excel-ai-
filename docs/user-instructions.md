# User Instructions

Persistent preferences for the AI agent. Updated via the `update_instructions` tool.

## Style
- Use Oxford commas in lists.
- Bold all sheet titles and section headers.

## Formatting
- Currency: millions with 1 decimal (e.g. 394.2)
- Percentages: 0 decimal places (e.g. 15%)
- Dates: YYYY-MM-DD

## Language
- Primary: Italian
- Technical terms: keep English (EBITDA, WACC, DCF)

## Defaults
- DCF projection years: 5
- Terminal growth rate: 2.0%
- Risk-free rate source: 10-year US Treasury

## Limitations
- Do not create PivotTables
- Do not use Power Query
