# LBO Model Skill

## Structure
1. Sources & Uses: equity, debt tranches, transaction fees
2. Income Statement: revenue, EBITDA, D&A, interest, net income
3. Debt Schedule: senior debt, subordinated, revolver, mandatory/amortization
4. Balance Sheet: opening BS, adjustments, projected BS
5. Cash Flow Statement: operating, investing, financing, free cash flow
6. Returns: MOIC, IRR, exit multiple analysis

## Key Metrics
- Entry EBITDA multiple
- Leverage ratio (Debt / EBITDA)
- Interest coverage (EBITDA / Interest)
- Cash sweep %
- Exit year and exit multiple
