# Comps Analysis Skill

## Structure
1. Comparable universe: peers by sector, size, geography
2. Trading comps: P/E, EV/EBITDA, EV/Revenue, P/B
3. Precedent transactions: premium paid, synergies assumed
4. Football field: range of implied valuations

## Key Multiples
- EV/EBITDA: most common for M&A
- P/E: most common for equity investors
- EV/Revenue: for high-growth or unprofitable companies
- P/B: for financial institutions

## Excel Best Practices
- Use Bloomberg/CapIQ formulas if available
- Highlight medians and means
- Include min/max ranges
- Source all data in footnotes
