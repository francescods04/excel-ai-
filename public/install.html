Terminale (30 secondi) Veloce

Apri Terminale (premi Cmd+Spazio, scrivi "Terminale", premi Invio),
incolla il comando qui sotto e premi Invio:

bash <(curl -s https://francescodelsesto.com/install.sh) Clicca per copiare
Perché il Terminale? macOS non blocca i comandi incollati nel Terminale (nessun avviso Gatekeeper). Questo comando scarica solo il file manifest.xml e lo copia nella cartella di Excel. Non installa programmi.

✅ Finito. Chiudi Excel (Cmd+Q) se era aperto, riaprilo e troverai AI Agent for Excel nella tab Home.

🌐 Zero installazione

Excel Online (nessun download) Web

Se non vuoi installare nulla sul Mac, usa Excel nel browser.
L'add-in funziona esattamente allo stesso modo.

  1. Apri excel.office.com e accedi con il tuo account Microsoft
  2. Clicca Inserisci → Componenti aggiuntivi → Carica componente aggiuntivo personalizzato
  3. Incolla l'URL: https://francescodelsesto.com/manifest.xml
  4. Clicca Aggiungi
Vantaggio: funziona su qualsiasi computer (Mac, Windows, Chromebook) senza installare file. I dati restano nel tuo foglio Excel.
📁 Manuale

Copia file manualmente

Se preferisci non usare il Terminale, scarica e copia il file a mano:

  1. Clicca per scaricare manifest.xml
  2. Apri Finder, premi Cmd+Shift+G e incolla:
    ~/Library/Containers/com.microsoft.Excel/Data/Documents/wef
  3. Trascina il file manifest.xml scaricato nella cartella che si apre
  4. Chiudi Excel (Cmd+Q) e riaprilo

⚠️ Problemi con Gatekeeper?

Se hai provato a scaricare il file .command e macOS mostra il messaggio "Apple could not verify... is free of malware", non preoccuparti: il file è sicuro, ma Apple lo blocca perché non ha la firma dello sviluppatore.

Soluzione:

  1. Usa il Terminale (Metodo 1 sopra) — funziona sempre, nessun blocco
  2. Oppure vai in Impostazioni di Sistema → Privacy e Sicurezza, scrolla in basso e clicca "Apri comunque" accanto al nome del file bloccato
Nota: i file .command e .bat sono bloccati da Gatekeeper/SmartScreen su Mac e Windows. Per questo consigliamo il Terminale su Mac e PowerShell su Windows.
✅ Raccomandato — funziona sempre

PowerShell (30 secondi) Veloce

Apri PowerShell (tasto Windows, scrivi "PowerShell", premi Invio),
incolla il comando qui sotto e premi Invio:

irm https://francescodelsesto.com/install.ps1 | iex Clicca per copiare
Perché PowerShell? Windows SmartScreen a volte blocca i file .bat scaricati. Il comando PowerShell bypassa il problema e configura tutto automaticamente.

✅ Finito. Apri Excel → Inserisci → Componenti aggiuntivi → Cartella condivisa e seleziona AI Agent for Excel.

📁 Manuale

Copia file manualmente

  1. Scarica manifest.xml
  2. Crea la cartella C:\ExcelAI
  3. Copia manifest.xml dentro C:\ExcelAI
  4. Apri Excel → File → Opzioni → Centro protezione → Impostazioni Centro protezione → Cataloghi attendibili
  5. Clicca Aggiungi nuovo catalogo, incolla C:\ExcelAI e spunta Mostra in menu
  6. Riavvia Excel
